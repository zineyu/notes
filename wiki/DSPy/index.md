# DSPy

> Stanford 提出的 LLM 编程框架，将 Prompt 工程转化为可优化的参数化程序。核心思想是用编译器/优化器替代手写 Prompt 调参。

## 页面
- [[BootstrapFewShot]] — 用 Teacher 模型自动筛选高质量 few-shot 示例
- [[MIPROv2]] — 联合优化指令和例题的贝叶斯优化方法
- [[GEPA]] — 基于反射变异的进化算法优化指令
